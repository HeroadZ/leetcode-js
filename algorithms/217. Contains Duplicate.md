# 217. Contains Duplicate

标签（空格分隔）： leetcode easy array

---

给一个整数数组，如果有重复，则返回true，否则返回false。

## 思路
我们用set来处理重复及查询问题。

## 代码
```js
/**
 * @param {number[]} nums
 * @return {boolean}
 */
var containsDuplicate = function(nums) {
    var mySet = new Set();
    for(var i=0;i<nums.length;i++) {
        if(mySet.has(nums[i])) return true;
        else mySet.add(nums[i]);
    }
    return false;
};
```
