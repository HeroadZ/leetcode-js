# Sum of Two Integers

标签（空格分隔）： leetcode

---

不用+和-计算两个整数的和。

## 思路

```md
其实二进制加法有点类似异或，只是异或缺少了进位，所以我们用与和左移1位表示进位。
3 0011
2 0010
——————
3^2得到0001，3&2得到0010，左移一位0100，异或得到0101，且没有再进位了。
```

## 代码

```js
/**
 * @param {number} a
 * @param {number} b
 * @return {number}
 */
var getSum = function(a, b) {
    return b === 0? a:getSum(a^b, (a&b)<<1);
};
```
